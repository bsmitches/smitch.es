# smitch.es

Personal website deployed at https://smitch.es

## Structure
- `/` - Work in Progress landing page
- `/wedding` - Wedding photo gallery with links to engagement, backyard wedding, and celebration photos

## Local Development
Open `index.html` in your browser to preview locally.

## Deployment
This site is deployed to GitHub Pages. Push to `main` branch to update the live site.

## Updating Gallery Links
Edit `wedding/index.html` and update the gallery link URLs as needed.
